游戏特色/Features
--------

* 可操控,当前最多可三个玩家同时操作
* 可保存


按键/Shortcuts
------

Esc: 暂停

1: 保存

2: 加载

4: 加入玩家

第一个玩家
w,a,s,d

第二个玩家
i,j,k,l

第三个玩家
上下左右
